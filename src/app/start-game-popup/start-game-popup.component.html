Single Player Game
Welcome New Recruit!

Win a single player game to unlock more game modes!

 Basic Training: You will be starting out as European Union.
Diplomacy: You can make alliances to form a 2v2 matchup. First team to control 6 capitals wins the game.
Free-For-All: You are on your own. First player to control 6 capitals wins the game.
All units are visible to all players.
You get approximate unit counts for each territory but not specific units.
Total fog blackout for all territories unless you have a unit that borders it.
